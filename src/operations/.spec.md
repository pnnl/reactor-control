# Operations Layer - Finalized Specification

---

## Module Structure

```
src/operations/
├── __init__.py
├── base.py                 # Base operation class with common patterns
├── temperature_control.py  # Thermal management operations
├── flow_control.py         # Gas and liquid flow operations
├── data_acquisition.py     # MKS ToolWEB data collection
├── sample_management.py    # Sample ID and material parameters
├── step_logger.py          # Comprehensive step logging
├── safety_interlocks.py    # Cross-device safety checks
└── config/
    ├── gas_calibrations/   # MFC calibration files per gas
    └── defaults.yaml       # Default operation parameters
    |── paths.yaml          # Default paths for data storage
```

## 1. `temperature_control.py`

### Parameters

| Parameter | Type | Exposed/Hidden | Default | Description |
|-----------|------|----------------|---------|-------------|
| `target_temps` | `list[float]` | Exposed | — | Temperature targets for each program step (°C) |
| `ramp_rates` | `list[float]` | Exposed | — | Heating/cooling rates (°C/min) per step |
| `hold_times` | `list[float]` | Exposed | — | Duration at each target (minutes) |
| `thermal_stabilization_time` | `float` | Exposed | 60.0 | Wait time (s) after reaching target before proceeding |
| `poll_interval` | `float` | Hidden | 5.0 | Temperature logging interval (seconds) |
| `tolerance` | `float` | Hidden | ±2.0 | Acceptable deviation from target (°C) |
| `timeout` | `float` | Hidden | 3600.0 | Max time to reach target before abort (s) |
| `overshoot_limit` | `float` | Hidden | 10.0 | Safety cutoff above target (°C) |

### Data Logging Strategy

**File 1: `temperature_log.csv`** (continuous, every 5 seconds)
```csv
datetime,target_temp,read_temp
2025-01-10T14:30:05,100.0,98.7
2025-01-10T14:30:10,100.0,99.2
...
```

**File 2: `readme.md`** (step summary for pretreatment record)
```markdown
# Thermal Pretreatment Steps

| Step ID | Target Temp (°C) | Ramp Rate (°C/min) | Hold Duration (min) |
|---------|------------------|--------------------|--------------------|
| 1       | 120              | 10                 | 30                 |
| 2       | 400              | 5                  | 60                 |
| 3       | 25               | natural            | 0                  |
```

## 4. `sample_management.py`

### Parameters (unchanged)

```python
@dataclass
class MaterialParameters:
    # Mandatory
    user_label: str           # Required - user-provided sample identifier
    material_type: str        # Required - e.g., "catalyst", "support", "standard"
    mass_mg: float            # Required - sample mass in milligrams
    
    # Optional
    operator: str | None = None           # Who is running the experiment
    composition: str | None = None        # e.g., "5% Pt/Al2O3"
    mesh_size: str | None = None          # e.g., "60-80 mesh"
    pretreatment_history: str | None = None
    batch_id: str | None = None
    notes: str | None = None
```

### Auto-generated sample_id
```
{YYYYMMDD}_{HHMMSS}_{material_type}_{user_label}
Example: 20250110_143052_catalyst_PtAl2O3_batch7
```

## 6. `safety_interlocks.py`

### Implementation

```python
@dataclass
class SafetyLimits:
    max_temperature: float = 1000.0      # °C
    max_total_flow: float = 500.0        # SCCM
    min_carrier_flow: float = 10.0       # SCCM (prevent stagnant atmosphere)
    max_ramp_rate: float = 50.0          # °C/min
    max_water_flow: float = 10.0         # mL/min

class SafetyInterlocks:
    def check_temperature(self, target: float) -> tuple[bool, str | None]
    def check_flow(self, total_sccm: float) -> tuple[bool, str | None]
    def check_all(self) -> tuple[bool, list[str]]
    def emergency_shutdown(self) -> None
```
