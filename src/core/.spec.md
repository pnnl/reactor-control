# Core Module Overview

## Purpose
Shared utilities and configuration for reactor control system.

## Components

### **Configuration** (`core/config.py`)
Centralized device settings and system configuration.

### **Logging**
Professional logging throughout the system.

## Current Settings
- **MFC Ports**: COM4, COM5 (9600 baud)
- **Thermocouple Port**: COM6 (115200 baud)
- **Connection Settings**: 3 retry attempts, 1 second delay
